# Contributing

1. Fork the repo and clone your fork.
2. Create a feature branch: `git checkout -b feature/my-change`.
3. Make changes and add tests/examples if applicable.
4. Run and self-check locally.
5. Commit with a clear message: `git commit -m "Explain what changed"`.
6. Push and open a Pull Request. Link the issue if any.
7. Follow the CODE_OF_CONDUCT.
